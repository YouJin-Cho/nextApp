# 🙌🏻 nextapp
 Next.js 생활코딩 강의를 보면서 핵심 기능에 대해 실습한 내용!
 <br><br>

## 👉🏻 Getting Started

```bash
1. clone(SSH) : git@github.com:YouJin-Cho/nextApp.git
2. terminal ➡️ cd nextApp
3. npm install
4. npm run dev
5. Open (http://localhost:3000)
```
